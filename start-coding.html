Typed Config

Start Coding with Tyco

Get up and running with Tyco in Python. Adjust commands for your environment as necessary.

Install

python -m pip install tyco

Basic usage

from tyco import load, loads

# Load from file
cfg = load('config.tyco')

# Load from string
config_text = '''
str app_name: MyApp
Server:
 *str name:
  - web1
'''
cfg = loads(config_text)

# Load entire directory recursively
cfg = load('config/')

print(cfg['Server'][0]['name'])

Run the CLI

# validate a file against the spec
tyco validate config.tyco

# validate entire directory
tyco validate config/

# dump a parsed JSON representation
tyco dump config.tyco --format json

# dump directory structure
tyco dump config/ --format json

Example config file

Create a config.tyco file:

str timezone: UTC

Application:
  str service:
  str profile:
  str host:
  - service: webserver, profile: primary, host: prod-01-us
  - service: database, profile: mysql, host: prod-02-us
